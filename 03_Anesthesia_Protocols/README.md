## Purpose
This folder contains three complete, evidence-informed anesthesia pathways for **post-mastectomy breast implant revision** surgery:

1. **Opioid-Free GA + PECS/Serratus Blocks** → Preferred  
2. **Opioid-Free GA without Regional Blocks** → Needle-minimizing  
3. **Conventional Opioid-Based GA** → Fallback  

All pathways emphasize multimodal analgesia, PONV reduction, hemodynamic stability, and ERAS principles while remaining practical for daily clinical use.

## Clinical Decision Guide

| Scenario | Recommended Pathway |
|----------|----------------------|
| Patient terrified of needles | **Protocol 2** (OFIA, no blocks) |
| High Apfel PONV score (≥3) | **Protocol 1** (with blocks) |
| Chronic opioid use / tolerance | **Protocol 3** or Protocol 1 |
| Suspected infection or extensive capsulectomy | **Protocol 1** (superior analgesia) |
| Difficult airway anticipated | Any pathway (all compatible with video laryngoscopy, etc.) |

**Bottom line:** Start with **Protocol 1** unless patient preference or contraindication dictates otherwise.
